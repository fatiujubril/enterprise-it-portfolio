### 🔐 Project 02 — Conditional Access Failure & Identity Attack Simulation
**Focus:** Identity misconfiguration, MFA bypass detection, incident response

A realistic identity security scenario demonstrating how a **Conditional Access exclusion** can silently disable MFA enforcement for a high-value executive account — without malware, phishing, or external attackers.

Key outcomes:
- Built a secure Conditional Access baseline
- Introduced a controlled misconfiguration (MFA exclusion)
- Detected MFA bypass via Entra ID sign-in telemetry
- Performed containment, remediation, and recovery
- Documented governance improvements to prevent recurrence
